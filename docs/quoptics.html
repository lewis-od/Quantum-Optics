quoptics.network module¶

class quoptics.network.NeuralNetwork(model_dir=None, dropout=0.2, learning_rate=0.001)¶

Bases: object

A tensorflow neural network for classifying quantum states of light

When inputting a state

\[\lvert \psi \rangle = \sum_{n=o}^{\infty} c_n \lvert n \rangle\]

where the \(\lvert n \rangle\) are Fock states, the network expects the state data to be in the form of a numpy array, where the \(n\) th element is the modulus of \(c_n\). The function to_numpy() converts qutip.Qobj instances to this form.

Parameters:
  • model_dir – The directory to save/load the model data to/from
  • dropout – The dropout probability to use when training
  • learning_rate – The learning rate to use when training

quoptics.states module¶

Methods for generating different types of state.

In all of the below text, \(\mathcal{N}\) is a normalisation constant, \(\widehat{S}(z)\) is the single-mode squeezing operator, and \(\lvert \alpha \rangle\) are coherent() states.
